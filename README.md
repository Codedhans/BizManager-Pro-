# 📊 BizManager OS (Pro)
**The Ultimate Offline Business Suite for Modern SMEs.**

BizManager OS is a lightweight, mobile-responsive management tool designed specifically for small business owners—from poultry farmers to fashion designers. Built with a "Zero-Internet" philosophy by **Hanson codedhans**, it ensures business data is always accessible, secure, and private.

## ✨ Key Features

-   **📦 Inventory Intelligence:** Track batches, stock levels, and maturity dates (perfect for livestock or limited collections).
-   **🧾 Visual Receipt Engine:** Generate professional, paper-style receipts that can be shared via WhatsApp or saved as PDFs.
-   **📣 Marketing Broadcast:** One-tap WhatsApp marketing to blast new arrivals or restock updates to your entire contact list.
-   **💰 Debt Ledger:** Dedicated tracking for customer balances with built-in reminder triggers.
-   **🌓 Persistent Dark Mode:** Full cross-page dark mode support for late-night business management.
-   **🛡️ Data Security:** LocalStorage-based data persistence with manual JSON backup and restore capabilities.

---

## 🛠️ Technical Stack

-   **Frontend:** HTML5, CSS3 (Custom Variables)
-   **Logic:** Vanilla JavaScript (ES6+)
-   **Storage:** Browser LocalStorage API
-   **Architecture:** Multi-page architecture with shared state via `script.js`.

## 🚀 How to Use

1.  **Clone the repo:**
    ```bash
    git clone [https://github.com/your-username/bizmanager-os.git](https://github.com/your-username/bizmanager-os.git)
    ```
2.  **Open the App:** Simply launch `index.html` in any modern mobile or desktop browser.
3.  **Settings:** Go to the Settings page to set your Business Name for custom branding on receipts.

## 📱 Mobile App (APK)
This project is optimized to be wrapped as a native Android application. The offline-first nature makes it feel like a native system app once installed, providing a smooth user experience without data costs.
